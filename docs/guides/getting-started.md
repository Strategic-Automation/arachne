# Getting Started: Installation and First Run

For a complete step-by-step tutorial, see [Getting Started Tutorial](../tutorials/getting-started.md).

This guide provides a quick overview:

1. **Clone**: `git clone https://github.com/Strategic-Automation/arachne`
2. **Install**: `uv sync`
3. **Configure**: Copy `.env.example` to `.env` and add your API keys
4. **Run**: `uv run arachne run "Your goal here"`

See [Configuration Reference](../configuration.md) for all available settings.
